Open Source, Effortless API Consumption, Asp.Net Dependency Injection, Integrated Mocking

SereneApi is a .NET Library intended to provide a straightforward way of consuming RESTful APIs requiring as little code as possible whilst providing a powerful set of extensible tools.

OVERVIEW

Delivering Effortless RESTful API Consumption, With an Extensible Suite of Tools

SereneApi was designed from the ground up with the intention of acting as middleware between your code and the API its consuming. Eliminiating the need to process requests & responses, whilst also offering you the ability to extend and supersede functionality.

"After using the repository pattern for many years I was inspired to create a generic API consumer that implemented similar concepts. I began fiddling with the idea in 2019 and after some tweaking I concluded the concept found and began working on SereneApi."

Bradley Clarke

ApiHandler Pattern

The ApiHandler pattern provides an abstraction layer between your application and the APIs its consuming. Safeguarding your application from API changes, easing maintenance and aiding in automated unit testing.

Dependency Injection

SereneApi fully supports Asp.Net Dependency Injection, consuming APIs has never been this easy.

Integrated Mocking

Mocking is fully supported for both API requests and responses, giving you the ability to perform 'end-to-end' without ever contacting the real thing. Granting easier than ever unit testing and promoting a faster and safer development lifecycle.
